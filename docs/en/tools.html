Changing the settings

To change the settings, click the Settings button. Then, the Settings page will open.
You can enable or disable autosaving by clicking on the "Autosave after" checkbox. To change the time (in seconds) before the database will autosave, edit the value inside the "Autosave after" field.
If a change is made before the database autosaves, the autosave will be delayed again.

Manually refreshing the one-time passwords

To manually refresh all one-time passwords, click the Refresh button. All one-time passwords are refreshed automatically, so you don't need to manually refresh them.
