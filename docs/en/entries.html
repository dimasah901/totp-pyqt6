Creating a new entry

To create a new entry, click the New entry... button.
You will be prompted to copy the QR code from the application you want to set up TOTP in. To do so, right click the QR code and choose Copy Image in the context menu. Then, click Yes at the bottom of the page. If the context menu does not open when you right click, simply click No. If the application will be able to recognize the QR code, all parameters will be auto-filled. After clicking Yes or No, the Edit entry page will open.

Editing an entry

After creating a new entry, or after highlighting an entry and clicking the Edit entry... button, the Edit entry page will open.
Here are the descriptions of all fields available:
Name The name of the entry that will be displayed on the main page in the Name column.
Secret The shared secret used to generate the one-time password. It is a base32-encoded string that consists of letters and digits. When setting up the TOTP, it is typically displayed under the QR code or hidden behind a button like "Can't scan it?" (Google) or "setup key" (GitHub).
Notes Your notes for the entry.

Advanced fields:
Algorithm The hashing algorithm used. Defaults to "SHA1", the most common algorithm used.
Time step The interval used to generate the one-time password. Defaults to 30 seconds as recommended in RFC 6238.
Origin time The starting time used to generate the one-time password. Defaults to 0, the Unix epoch.
Code size The length of the one-time password that is generated. Defaults to 6. Accepts values more than or equal to 1 and less than or equal to 10.

Deleting an entry

After highlighting an entry and clicking the Delete entry... button, the entry will be irreversibly deleted (unless you prevent the database from auto-saving).

Copying an entry

After highlighting an entry and clicking the Copy TOTP button, the current one-time password will be copied to the clipboard.
