Creating a new database

To create a new database, click the New database... button.
You will be prompted to choose the file that the database will be saved to and asked to enter a new password. If your password only has digits, it must be at least 28 digits long. If the password has letters, it must be at least 16 characters long.

Saving the database

To save the database, click the Save database... button.

To save the database under a new file, click the Save database as... button.
You will be prompted to choose the file that the database will be saved to. After choosing the file, any new changes will be written to the new file instead of the old one.

To save a copy of the database, click the Save database backup... button.
You will be prompted to choose the file that the database will be copied to. After choosing the file, no new changes will be written to the new file unless you choose to save a copy of the database again.

Closing the database

To close the database, click the Close database button.
