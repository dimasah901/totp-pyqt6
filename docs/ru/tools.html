Изменение настроек

Для изменения настроек выберите пункт Настройки.
Вы можете включить или выключить автосохранение с помощью галочки "Автосохранить после". Для изменения задержки (в секундах) сохранения базы данных, измените значение в поле "Автосохранить после".
Если до автосохранения базы данных будут внесены изменения, автосохранение будет отложено.

Вручную обновить все одноразовые пароли

Для обновления всех одноразовых паролей вручную выберите пункт Обновить. Все одноразовые пароли обновляются автоматически.
