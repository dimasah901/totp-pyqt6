Создание новой базы данных

Для создания новой базы данных выберите пункт Новая база данных...
Вам будет предложено выбрать файл, в который будет сохранена база данных, и ввести новый пароль. Если ваш пароль состоит только из цифр, он должен содержать не менее 28 цифр. Если пароль содержит буквы, он должен быть длиной не менее 16 символов.

Сохранение базы данных

Для сохрания базы данных выберите пункт Сохранить базу данных...

Для сохрания базы данных в новом файле выберите пункт Сохранить базу данных как...
Вам будет предложено выбрать файл, в котором будет сохранена база данных. После выбора файла все новые изменения будут записываться в новый файл вместо старого.

Для сохрания копии базы данных выберите пункт Сохранить копию базы данных...
Вам будет предложено выбрать файл, в который будет скопирована база данных. После выбора файла новые изменения не будут записаны в новый файл, пока вы не решите снова сохранить копию базы данных.

Закрытие базы данных

Для закрытия базы данных выберите пункт Закрыть базу данных.
