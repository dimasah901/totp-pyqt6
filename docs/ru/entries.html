Создание новой записи

Для создания новой записи выберите пункт Новая запись...
Вам будет предложено скопировать QR-код из приложения, в котором вы хотите настроить TOTP. Чтобы это сделать, нажмите правой кнопкой мыши по QR-коду и выберите Скопировать картинку во всплывающем окне. Затем, выберите внизу страницы Да. Если при нажатии правой кнопкой мыши ничего не появляется, нажмите Нет. Если приложению удастся отсканировать QR-код, все параметры будут заполнены автоматически. После нажатия Да или Нет откроется страница Редактировать запись.

Редактирование записи

После создания новой записи или выделения существующей и нажатия пункта Редактировать запись... откроется страница Редактировать запись. Вот описания всех доступных полей:
Название Название записи, которое будет отображаться в столбце Название.
Секрет Общий секрет, используемый для генерации одноразового пароля. Это закодированная в base32 строка, состоящая из букв латинского алфавита и цифр. При настройке TOTP данный секрет обычно отображается под QR-кодом или находится за кнопкой, подобной "Не удается отсканировать?" (Google) или "ключ настройки" (GitHub).
Заметка Ваша заметка для текущей записи.

Расширенные настройки:
Алгоритм Используемый хеш-алгоритм. По умолчанию используется "SHA1" - самый распространенный хеш-алгоритм.
Временной шаг Интервал, используемый для генерации одноразового пароля. По умолчанию используется интервал в 30 секунд, по рекомендации RFC 6238.
Время начала отсчета Время начала отсчета, используемое для генерации пароля. По умолчанию используется 0, начало отсчета Unix времени.
Длина пароля Длина генерируемого одноразового пароля. По умолчанию используется 6. Принимает значения от 1 до 10 включительно.

Удаление записи

После выделения записи и выбора пункта Удалить запись..., запись будет безвозвратно удалена (если вы не предотвратите автосохранение базы данных).

Копирование записи

После выделения записи и выбора пункта Скопировать TOTP, текущий одноразовый пароль будет скопирован в буфер обмена.
